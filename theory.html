Theoretical Foundation

Understanding Linear Dependence and Independence

Basic Concepts

Linear dependence and independence are fundamental concepts in linear algebra that describe relationships between vectors in a vector space. These concepts help us understand whether vectors provide unique information or if some vectors are redundant combinations of others.

Intuitive Understanding

Linear Independence: A set of vectors is linearly independent if no vector in the set can be written as a linear combination of the others. Each vector adds a new "direction" or dimension.

Linear Dependence: A set is linearly dependent if at least one vector can be expressed as a linear combination of the others. This means there's redundancy in the information provided by the vectors.

Vector Space: A collection of vectors that can be added together and multiplied by scalars (real numbers) while satisfying certain axioms.
Linear Combination: An expression constructed from a set of vectors by multiplying each vector by a scalar and adding the results.
Span: The set of all possible linear combinations of a given set of vectors. The span of linearly independent vectors forms a basis for a subspace.

Formal Definitions

Definition 1: Linear Combination

Given vectors v₁, v₂, ..., vₖ in a vector space V over a field F (typically ℝ), a linear combination of these vectors is any vector of the form:

c₁v₁ + c₂v₂ + ... + cₖvₖ
where c₁, c₂, ..., cₖ ∈ F are scalars.
Definition 2: Linear Dependence

A set of vectors {v₁, v₂, ..., vₖ} is linearly dependent if there exist scalars c₁, c₂, ..., cₖ, not all zero, such that:

c₁v₁ + c₂v₂ + ... + cₖvₖ = 0
where 0 is the zero vector.
Definition 3: Linear Independence

A set of vectors {v₁, v₂, ..., vₖ} is linearly independent if the only solution to:

c₁v₁ + c₂v₂ + ... + cₖvₖ = 0
is c₁ = c₂ = ... = cₖ = 0.

This is called the trivial solution.

Important Theorems

Theorem 1
Criterion for Linear Dependence

A set of vectors {v₁, v₂, ..., vₙ} in ℝⁿ is linearly dependent if and only if the determinant of the matrix formed by these vectors as columns (or rows) is zero.

det([v₁ v₂ ... vₙ]) = 0 ⇔ {v₁, v₂, ..., vₙ} is linearly dependent
Theorem 2
Rank and Linear Independence

The rank of a matrix A equals the maximum number of linearly independent column vectors (or row vectors) of A. For a set of m vectors in ℝⁿ:

rank(A) = number of linearly independent vectors

Vectors are linearly independent if and only if rank(A) = m (number of vectors).

Theorem 3
Basis and Dimension

A set of vectors in a vector space V is a basis for V if:

  1. The vectors are linearly independent
  2. The vectors span V (every vector in V can be written as their linear combination)

The dimension of V is the number of vectors in any basis for V.

Detection Methods

Method 1: Gaussian Elimination (Row Reduction)

The most general method for determining linear dependence/independence involves creating a matrix with the vectors as columns and performing Gaussian elimination to find its row echelon form.

Step 1: Form matrix A with given vectors as columns
Step 2: Reduce A to row echelon form using elementary row operations
Step 3: Count the number of pivot positions (non-zero rows)
Step 4: Compare rank with number of vectors
• If rank = number of vectors → Linearly Independent
• If rank < number of vectors → Linearly Dependent
Method 2: Determinant Method (for Square Matrices)

For n vectors in ℝⁿ (square matrix), compute the determinant:

• det ≠ 0 → Vectors are linearly independent
• det = 0 → Vectors are linearly dependent
Example

For vectors v₁ = (1, 2), v₂ = (3, 4) in ℝ²:

Matrix A = [v₁ v₂]
1 3
2 4

det(A) = (1×4) - (3×2) = 4 - 6 = -2 ≠ 0

Conclusion: v₁ and v₂ are linearly independent.

In ℝ² (2D Plane)

• Two vectors: Linearly dependent if they lie on the same line through the origin
• Three or more vectors: Always linearly dependent (ℝ² has dimension 2)

Examples in ℝ²

Independent: v₁ = (1,0), v₂ = (0,1) → Form a coordinate system

Dependent: v₁ = (1,2), v₂ = (2,4) → v₂ = 2·v₁ (collinear)

In ℝ³ (3D Space)

• Two vectors: Independent unless they're collinear
• Three vectors: Dependent if they lie in the same plane through the origin
• Four or more vectors: Always linearly dependent (ℝ³ has dimension 3)

Examples in ℝ³

Independent: (1,0,0), (0,1,0), (0,0,1) → Standard basis for ℝ³

Dependent: (1,2,3), (2,4,6), (3,6,9) → All are scalar multiples
